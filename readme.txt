== Description ==

This plugin allows you to add your own description for each plugin in plugins list page.
Sometimes we install a plugin. but after a while, we forget its main functionality.
Adding custom description for each plugin let us prevent this kind of forgetting.


== Installation ==

Upload the plugin and activate it.

1, 2, 3: You're done!
